Web Project Plan – CineViaggio

1. Brief di progetto

Titolo progetto: CineViaggio

Obiettivo: Creare una piattaforma web che permetta di esplorare le location cinematografiche iconiche in Italia (e nel mondo) attraverso una mappa interattiva, schede dettagliate e contenuti visuali.

Target: Appassionati di cinema, viaggiatori, studenti DAMS, turisti, studiosi di media.

Funzionalità principali:

  • Navigazione tra i film per genere, regione, città
  • Mappa interattiva dei luoghi di ripresa
  • Gallerie fotografiche
  • Ricerca avanzata
  • Trame e dettagli

3. Struttura del sito

  • Home: introduzione, film in evidenza, accesso alle sezioni
  • Collezione: elenco di tutti i film, filtrabile
  • Scheda film: dettagli, mappa, immagini
  • Mappa: esplora tutti i film tramite localizzazione
Home
├── Collezione

├── Mappa

5. Scelte di design e usabilità

  • Colori: palette cinema-classico (nero, rosso, oro)
  • Font: Arial / Roboto per leggibilità
  • Layout: responsive con Bootstrap
  • Accessibilità: uso di alt, buon contrasto, struttura chiara

6. Servizi e Tools usati

  • Leaflet.js per la mappa
  • Bootstrap per la UI
  • JavaScript per il caricamento dinamico
  • JSON come base dati
  • Dublin Core per i metadati
  • Filtri e ricerca per il browsing
  • Integrazioni in corso: Google Analytics, breadcrumbs, accordion
